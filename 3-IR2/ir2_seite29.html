Übung 13: Indexierung und Retrieval

Diese Übung besteht aus 3 Teilaufgaben, welche in den folgenden Seiten bearbeitet werden sollen.

Aufgabe 1:

Indexieren Sie die Dokumente, die sie als Kurzfassungen auf der nächsten Seite finden. Gehen Sie dabei wie ein automatisches System vor, das Stemming und Kompositazerlegung kann, und verwenden Sie das kontrollierte Vokabular (die Deskriptoren werden vorgegeben).

1.1. Stellen Sie eine Dokument-Term-Matrix (DTM) und die Anfrage-Vektoren auf.
1.2. Zählen Sie die Term-Frequenz jedes Terms in jedem Dokument.
1.3. Berechnen Sie die Inverse Dokumenthäufigkeit (idf) mit der vereinfachten Formel:
idf = (Anzahl d. Dok. in der Kollektion) / (Anzahl d. Dok. mit dem Term)

Aufgabe 2:

Berechnen Sie die Gewichtungsvektoren für alle Dokumente und Anfragen.

Aufgabe 3:

Berechnen Sie die Ähnlichkeit zwischen Anfragen und Dokumenten. Nutzen Sie dazu den Dice'sche Koeffizienten. Stellen Sie ein Ranking auf.
